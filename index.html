TrackOBotty

Free iOS companion app for Track-o-Bot.com

Update:This app is now available on the App Store

TrackOBotty is a universal iOS app that allows you to manually upload games to Track-o-Bot. It is ideal if you want to track your game results when playing Hearthstone on a iPad or iPhone, especially if you are already using TrackOBot on your Windows PC or Mac.

While you can create a new new account directly in the app, it is also possible to reuse your existing Track-o-Bot account. To do so, after installing this app, export your account data from your Desktop client ("Settings"->"Account"->"Export"), and then open the resulting file on your iOS device. This can be accomplished, for example, by sending the account data file via email to yourself, and opening the attachment on the iOS device. Another way would be using Dropbox or similar services.

The app has the following features:

  • Import account data file (Track-o-Bot profile credentials)
  • Add games with information about heroes, decks, coin, mode, rank and game results
  • Basic History view, allows to delete games
  • Win rate graphs for classes and decks
  • Open the trackobot.com web profile, export the profile credentials

That's it ;-) Obviously, since its free and Open Source, it comes with no warranties, use at your own risk (and see License information below).
