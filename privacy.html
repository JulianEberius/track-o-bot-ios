Privacy information

Track-o-Botty does not collect any user data, except for the game results and login credentials entered manually by the user. The login data is stored on the device only for the purpose of authenticating with the trackobot.com API, and can be exported and deleted (by logging out) in the "More" tab. The game results are not stored on the device, but are sent directly to the trackobot.com API. The user data retrieved from this API, such as result history and result statistics, are only used for the purpose of display in this app, and are never stored or used in any other context.   Note that Track-o-Botty is developed independently of the original Track-o-Bot software, and is using the TrackOBot API to store and retrieve user data from https://trackobot.com
